Recharge Transcash en ligne

Sur Recharge.fr vous pouvez acheter des coupons Transcash pour recharger votre carte Transcash d'une valeur de 20 €, 50 € ou 100 €. Il suffit de sélectionner le montant que vous souhaitez et de régler sur la plateforme de paiement sécurisée : par carte bancaire ou avec un compte PayPal. Par exemple, sélectionnez la recharge Transcash de 50 € et recevez ensuite le code de votre recharge par e-mail.

Un ticket Transcash est une carte prépayée rechargeable en ligne. Cette carte vous permet de faire des achats en ligne ou en magasins comme une carte bancaire ordinaire. Vous pouvez acheter une carte Transcash dans 35 000 points de vente en France ou en ligne sur le site internet de Transcash. Avec votre carte prépayée, il est possible de faire des achats chez les commerçants, de retirer de l’argent, de faire des achats sur internet, transférer de l’argent ou encore de verser en toute sécurité de l'argent vos proches.

Notez que la carte "Transcash Visa" n'est plus disponible, dû à un changement de programme. Vous recevrez à présent une carte prépayée Transcash Mastercard.



Astuce vigilance en ligne :
Nous vous recommandons de ne jamais partager vos codes Transcash par e-mail, SMS, sur les réseaux sociaux, ou via un site de vérification en ligne. Chargez votre code directement et uniquement sur le site officiel de Transcash.

Comment utiliser mon coupon Transcash ?


Information importante : Les codes Transcash vendus sur Recharge.fr ne sont pas des cartes prépayées, mais des recharges pour votre carte Transcash. Cela veut dire qu’il faut au préalable déjà avoir une carte Transcash Mastercard.

Si vous vivez en France, vous pouvez commander une carte sur le site officiel de Transcash.


Vous ne pensiez pas acheter ce code aujourd’hui? Faîtes preuve de prudence quant aux arnaques courantes avec ce produit. Ne partagez ce code avec personne d’autre que Transcash, sur leur site officiel www.transcash.fr.


Comment recharger votre carte:
Le code de Transcash que vous avez reçu recharge votre carte Transcash en suivant les étapes suivantes:


  1. Connectez-vous à votre espace personnel Transcash

  2. Allez à la section Rechargement

  3. Entrez votre code de recharge Transcash

  4. Votre carte est alors rechargée et utilisable pour son montant actuel.


Pour pouvoir charger un coupon Transcash vous devez être porteur d'une carte Transcash.

Pourquoi dois-je payer des frais de service supplémentaires pour Transcash ?


Pour Transcash, vous devez payer les frais de service Transcash, et les frais de service recharge.fr en plus. Tu payes toujours 21,50 € pour 20 € de crédit.

Puis-je acheter un ticket Transcash avec PayPal ?


Sur Recharge.fr, vous pouvez acheter Transcash et payer avec PayPal. Sélectionnez simplement le mode de paiement PayPal lorsque vous êtes sur la page de règlement. Commandez en toute discrétion et sécurité avec un compte PayPal, et recevez les codes de votre Transcash par e-mail en 30 secondes.

Comment contrôler le solde de mon ticket Transcash ?


Vous pouvez consulter le solde disponible sur votre ticket Transcash 7 jours sur 7 et 24 heures sur 24 en vous connectant à votre compte Transcash en ligne.

Si vous préférez connaître le montant par téléphone, vous pouvez joindre le service client Transcash au 0 826 666 555 (0,15€/ mn).

Quelle est la durée de validité de mon coupon Transcash ?


Si vous avez rechargé votre carte Transcash avec le crédit de votre ticket Transcash, vous pouvez utiliser ce crédit jusqu'à la date d'expiration de votre carte (indiquée sur la carte de paiement prépayée Transcash). Le code est valable 1 an.

Puis-je utiliser mon code Transcash en plusieurs fois ?


Votre ticket Transcash vous permet de recharger votre carte Transcash une fois. Le montant est ensuite sur la carte et vous pouvez utiliser celle-ci autant de fois que vous voulez jusqu'à l'expiration de votre crédit Transcash. Rechargez de nouveau votre carte Transcash en achetant un nouveau coupon.

Où puis-je utiliser ma carte Transcash ?


En ligne : Sur tout site internet qui accepte Mastercard comme méthode de paiement.

En magasins :
Votre carte fonctionne comme une Mastercard normale. Selon votre formule, vous pouvez utiliser votre carte en France et DOM-TOM, ou à l'étranger.

Comment contacter le service client Transcash ?


Par e-mail : service-client@trans-cash.fr

Par téléphone : 0826 666 555 (0,15€/MIN +prix d'un appel)

Si vous êtes ici parce qu'une personne que vous ne connaissez pas vous a demandé d'acheter ce code, nous vous recommandons d'être très vigilants. Les fraudeurs en ligne se font souvent passer pour des personnes ou organisations de confiance. Nous ne pourrons malheureusement pas vous aider s'ils mettent leur main sur votre code.
